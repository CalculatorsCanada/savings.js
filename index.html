cc-savings.js is a tiny JavaScript library for calculating simple or compound interest rate.

Check out the cc-savings.js homepage for demos and documentation!

Please checkout/download the latest stable tag before using in production. Bug reports and pull requests hugely welcome.

Made with love by @CalculatorsCa and some excellent contributors.
